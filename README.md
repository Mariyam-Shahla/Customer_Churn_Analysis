# Customer Churn Analysis

## Overview
This project focuses on analyzing customer churn data using Power BI to identify factors affecting customer retention and business performance through interactive dashboards and visual analytics.

## Objectives
- Analyze customer churn patterns
- Identify factors influencing customer retention
- Track customer demographics and subscription behavior
- Create an interactive business dashboard

## Tools & Technologies Used
- Power BI
- Data Visualization

## Dataset Information
The dataset contains customer information including demographics, subscription details, payment methods, contract types, and churn status.

## Project Workflow
1. Data Collection
2. Data Cleaning & Transformation
3. Data Modeling
4. Dashboard Creation
5. KPI Analysis
6. Insight Generation

## Key Analysis Performed
- Customer churn rate analysis
- Contract type comparison
- Monthly revenue analysis
- Customer demographics analysis
- Subscription and payment behavior analysis

## Key Insights
- Customers with monthly contracts showed higher churn rates
- Certain payment methods were associated with increased customer churn
- Long-term contract customers had better retention rates

## Dashboard Features
- Interactive filters and slicers
- KPI cards
- Churn trend visualizations
- Customer segmentation analysis
- Revenue insights dashboard
